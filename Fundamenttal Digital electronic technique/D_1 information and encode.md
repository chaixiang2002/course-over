### **补码的诞生**
解决原码 ‘0’的表示问题 以及 源码难相加  等问题  
        原码：
```

    +5：0 101   相加==  0 010     
    -5：1 101   等于 ‘2’ 而不是0
```
补码：  
```
    +5：0 0101  +5+（-5）=0 0000
    -5：1 1011
```
补码规则:  
- 正数不变
- 负数，数值位求反，再加1  


8421码和余3码：  
余3码解决了8421码进位困难的问题：  
8421：
```
    9:  1001
    2:  0010
    9+2:1011    并没有进位
```
余3码：  
```
    9:  1100
    2:  0100
    9+2:0000 进了1
```
